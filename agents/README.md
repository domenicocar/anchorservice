# Project Agents

This folder contains specialized project agents and recurring reviewer checklists for the Anchor Service website.

Use these files when a task needs a focused perspective, such as design consistency, SEO, accessibility, content, forms or deployment.

Each agent should define:

- purpose
- when to use it
- responsibilities
- checks to perform
- constraints
- expected output

Current agents:

- `anchor-design-reviewer.md` - design system, visual consistency, accessibility and brand-fit review.
- `anchor-seo-reviewer.md` - SEO, local SEO, metadata, schema, indexability and content-quality review.
